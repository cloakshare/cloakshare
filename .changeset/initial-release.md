Initial release of @cloakshare/viewer and @cloakshare/react.

- Web Component `<cloak-viewer>` with Shadow DOM encapsulation
- Client-side PDF rendering (lazy-loaded PDF.js) and image rendering (Canvas API)
- API-connected mode for Office docs and video
- Watermark engine with {{email}}, {{date}}, {{session_id}} templates
- Email gate with localStorage persistence
- Password protection (client-side and server-verified)
- Link expiry with server time mitigation
- Dark and light themes
- Mobile pinch-to-zoom and swipe navigation
- Keyboard navigation and ARIA accessibility
- Print, right-click, and drag protection
- React wrapper with typed props, events, and ref forwarding
